PORTFOLIO

OOP DATABASE

During this sprint, my team and I set out to build an application to display a biscuit database using Object Oriented PHP, HTML and CSS. The functionality includes: viewing more details on any selected biscuit, voting on your favourite biscuits (which are then saved in the session) and displaying the most/least popular biscuits. All functions were unit tested and custom error handling was used throughout.
